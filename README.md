# Moegi-theme
Moegi「萌黄」 - 一个Ghost主题
A simple theme for [Ghost](http://github.com/tryghost/ghost/).

## Style

主题名字的来源取自[NIPPON COLORS - 日本の伝統色](http://nipponcolors.com)。这是一个相当文艺的取色网站，收录的几百种颜色全部是日本的传统颜色，十分耐看，同时名字也非常好听。比如==一斥染==、==柳染==、==利休茶==等等。强烈建议需要找颜色的话来这里看看~
